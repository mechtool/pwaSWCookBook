Сетевые уведомления

Этот группирующий блок содержит общие подходы в использовании сервисного рабочего. В нем используется единая основа состоящая из одного фрейма, в который загружается HTML документ, соответствующий определенному примеру. Такая структура позволяет загружать своего сервисного рабочего в отдельный документ, избегая неразберихи сервисных рабочих при отладке.
